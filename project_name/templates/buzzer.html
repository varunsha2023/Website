Arduino Blinking Buzzer Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to make a buzzer produce sound using Arduino.

Introduction:

In this experiment, we will use an Arduino microcontroller to control the buzzing of a buzzer. This serves as an introduction to using digital output to control external devices like buzzers.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • Buzzer
  • Jumper wires
  • Breadboard

Code:

Below is the Arduino code to make a buzzer produce sound:

            
                // Define the pin connected to the buzzer
                int buzzerPin = 13;

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the digital pin as an output
                    pinMode(buzzerPin, OUTPUT);
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Turn the buzzer on (HIGH) for 1 second
                    digitalWrite(buzzerPin, HIGH);
                    delay(1000);

                    // Turn the buzzer off (LOW) for 1 second
                    digitalWrite(buzzerPin, LOW);
                    delay(1000);
                }
