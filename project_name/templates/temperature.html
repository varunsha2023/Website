Arduino Temperature Sensor Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to read temperature using a temperature sensor with Arduino.

Introduction:

In this experiment, we will use an Arduino microcontroller to read the temperature from a temperature sensor. This is a common task in many projects involving environmental monitoring, home automation, and more.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • Temperature sensor (e.g., TMP36)
  • Resistor (if required by the temperature sensor)
  • Jumper wires
  • Breadboard

Code:

Below is the Arduino code to read temperature from the sensor:

            
                // Define the analog pin connected to the temperature sensor
                int tempPin = A0;

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize serial communication at 9600 baud rate
                    Serial.begin(9600);
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Read the analog value from the temperature sensor
                    int sensorValue = analogRead(tempPin);

                    // Convert the analog value to voltage
                    float voltage = sensorValue * (5.0 / 1023.0);

                    // Convert voltage to temperature in degrees Celsius
                    float temperatureC = (voltage - 0.5) * 100;

                    // Output the temperature value to the serial monitor
                    Serial.print("Temperature: ");
                    Serial.print(temperatureC);
                    Serial.println("Â°C");

                    // Delay for a short interval before reading again
                    delay(1000);
                }
