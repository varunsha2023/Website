Arduino PIR Sensor Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to use a PIR (Passive Infrared) motion sensor with Arduino.

Introduction:

In this experiment, we will use a PIR motion sensor to detect motion in its field of view. PIR sensors are commonly used to detect motion by measuring changes in infrared radiation levels.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • PIR motion sensor
  • Jumper wires

Code:

Below is the Arduino code to detect motion using a PIR sensor:

            
                // Define the pin connected to the PIR sensor
                int pirPin = 2; // You can change this pin according to your setup

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the serial communication
                    Serial.begin(9600);

                    // Initialize the PIR pin as an input
                    pinMode(pirPin, INPUT);
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Read the state of the PIR sensor
                    int motionDetected = digitalRead(pirPin);

                    // Print the state to the Serial Monitor
                    if (motionDetected == HIGH) {
                        Serial.println("Motion detected!");
                    } else {
                        Serial.println("No motion detected.");
                    }

                    // Delay before next reading
                    delay(1000);
                }
