Arduino PIR Sensor with LCD Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to use a PIR (Passive Infrared) motion sensor with an LCD display using Arduino.

Introduction:

In this experiment, we will use a PIR motion sensor to detect motion and display the status on an LCD display. PIR sensors are commonly used to detect motion by measuring changes in infrared radiation levels.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • PIR motion sensor
  • LCD display (16x2)
  • Jumper wires

Code:

Below is the Arduino code to detect motion using a PIR sensor and display the status on an LCD:

            
                #include <LiquidCrystal.h>

                // Define the pins connected to the LCD
                LiquidCrystal lcd(12, 11, 5, 4, 3, 2); // Change the pin numbers according to your setup

                // Define the pin connected to the PIR sensor
                int pirPin = 7; // You can change this pin according to your setup

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the LCD display with 16 columns and 2 rows
                    lcd.begin(16, 2);

                    // Initialize the PIR pin as an input
                    pinMode(pirPin, INPUT);

                    // Set up the initial display message
                    lcd.setCursor(0, 0);
                    lcd.print("PIR Sensor:");
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Read the state of the PIR sensor
                    int motionDetected = digitalRead(pirPin);

                    // Display the motion detection status on the LCD
                    lcd.setCursor(0, 1);
                    if (motionDetected == HIGH) {
                        lcd.print("Motion Detected");
                    } else {
                        lcd.print("No Motion");
                    }

                    // Delay before next reading
                    delay(1000);
                }
