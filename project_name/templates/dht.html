Arduino DHT11 Sensor Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to use a DHT11 temperature and humidity sensor with Arduino.

Introduction:

In this experiment, we will use a DHT11 sensor to measure temperature and humidity. The DHT11 sensor is a basic, low-cost digital temperature and humidity sensor that is commonly used in Arduino projects.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • DHT11 temperature and humidity sensor
  • Jumper wires

Code:

Below is the Arduino code to read temperature and humidity using a DHT11 sensor:

            
                #include <DHT.h>

                // Define the type of sensor and the pin it's connected to
                #define DHTPIN 7 // Change this pin according to your setup
                #define DHTTYPE DHT11 // Change this to DHT22 if you're using DHT22 sensor

                // Initialize the DHT sensor
                DHT dht(DHTPIN, DHTTYPE);

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the serial communication
                    Serial.begin(9600);

                    // Initialize the DHT sensor
                    dht.begin();
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Read temperature and humidity from the sensor
                    float temperature = dht.readTemperature();
                    float humidity = dht.readHumidity();

                    // Check if any reads failed and exit early (to try again)
                    if (isnan(temperature) || isnan(humidity)) {
                        Serial.println("Failed to read from DHT sensor!");
                        return;
                    }

                    // Print temperature and humidity to Serial Monitor
                    Serial.print("Temperature: ");
                    Serial.print(temperature);
                    Serial.println(" Â°C");
                    Serial.print("Humidity: ");
                    Serial.print(humidity);
                    Serial.println(" %");

                    // Delay before next reading
                    delay(2000);
                }
