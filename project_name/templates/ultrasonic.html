Arduino Ultrasonic Sensor Tutorial

Aim of the Experiment:

The aim of this experiment is to learn how to use an ultrasonic sensor to measure distance using Arduino.

Introduction:

In this experiment, we will use an ultrasonic sensor to measure distance. Ultrasonic sensors work by emitting sound waves and measuring the time it takes for the waves to bounce back after hitting an object. This information can be used to calculate the distance to the object.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • Ultrasonic sensor (e.g., HC-SR04)
  • Jumper wires
  • Breadboard

Code:

Below is the Arduino code to measure distance using an ultrasonic sensor:

            
                // Define the pins connected to the ultrasonic sensor
                int trigPin = 7; // Trigger pin
                int echoPin = 6; // Echo pin

                // Variables to hold the duration of the pulse
                long duration;
                int distance;

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the serial communication
                    Serial.begin(9600);

                    // Initialize the trigPin as an output
                    pinMode(trigPin, OUTPUT);
                    // Initialize the echoPin as an input
                    pinMode(echoPin, INPUT);
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Clear the trigPin
                    digitalWrite(trigPin, LOW);
                    delayMicroseconds(2);

                    // Send a pulse to the sensor
                    digitalWrite(trigPin, HIGH);
                    delayMicroseconds(10);
                    digitalWrite(trigPin, LOW);
