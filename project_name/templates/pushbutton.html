Arduino Blinking LED Tutorial with Push Button

Aim of the Experiment:

The aim of this experiment is to learn how to make an LED blink using Arduino controlled by a push button.

Introduction:

In this experiment, we will use an Arduino microcontroller to control the blinking of an LED based on the state of a push button. Pressing the button will toggle the LED on and off.

Components Required:

  • Arduino board (e.g., Arduino Uno)
  • LED
  • Resistor (220 ohms)
  • Push button
  • Jumper wires
  • Breadboard

Code:

Below is the Arduino code to blink an LED with a push button:

            
                // Define the pin connected to the LED
                int ledPin = 13;
                // Define the pin connected to the push button
                int buttonPin = 2;
                // Variable to store the current state of the button
                int buttonState = 0;

                // Setup function runs once when the sketch starts
                void setup() {
                    // Initialize the digital pin as an output for the LED
                    pinMode(ledPin, OUTPUT);
                    // Initialize the digital pin as an input for the push button
                    pinMode(buttonPin, INPUT);
                }

                // Loop function runs repeatedly as long as the Arduino is powered on
                void loop() {
                    // Read the state of the push button
                    buttonState = digitalRead(buttonPin);

                    // If the button is pressed, toggle the LED
                    if (buttonState == HIGH) {
                        // Toggle the LED state
                        digitalWrite(ledPin, !digitalRead(ledPin));
                        // Wait a short delay to debounce the button
                        delay(100);
                    }
                }
